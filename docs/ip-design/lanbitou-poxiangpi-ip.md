# 烂笔头和破橡皮 IP 设定

## 定位

烂笔头和破橡皮是一对用于中文文章正文配图的低微、丑萌、抽象工具型 IP。

它们不是儿童文具拟人，也不是可爱吉祥物，而是被用残了、还在认真工作的创作工具残件。它们适合出现在写作、思考、复盘、AI 工作流、方法论、知识生产和认知校准类内容中。

核心关系：

> 烂笔头负责先错出来，破橡皮负责别错太远。

或者：

> 一个制造痕迹，一个制造空白。

## 角色一：烂笔头

### 核心人格

莽、急、乐观、先动手。

烂笔头相信“先画出来才知道错在哪”。它不怕写烂，怕空着。它的低微感来自：短到快握不住了，笔尖还容易断，但仍然冲在第一个。

### 外形

- 几乎用完的短铅笔头。
- 身体细、尖、斜，常常前倾。
- 黄色漆面脏旧、刮花、掉漆。
- 木头削得参差不齐。
- 前端笔芯钝、歪、容易断。
- 两条很细的小腿，站不稳但老往前冲。
- 小点眼，表情固执、空、认真。
- 尾部有明显的大金属包圈，包圈有 2-3 道压痕。
- 包圈里夹着快用完的残橡皮头。
- 残橡皮头只能擦小错，不能替代破橡皮的修正职责。

### 动作

- 划第一条歪线：开始、破局。
- 戳破纸面：追问、突破表面。
- 把乱线圈起来：聚焦、定义问题。
- 用身体当路标：给下一步方向。
- 拖着铅笔屑：过程消耗、长期积累。
- 写到一半断芯：思路卡住。
- 被削得更短：迭代代价。
- 给乱稿接上线：承接上下文。
- 在边角写小字：补充、旁注、脚注。
- 用尾部残橡皮头蹭一下：临时补救、边写边改。

### 口头禅

- “先写。”
- “有一笔再说。”
- “错了再说。”
- “空着才最糟。”
- “别擦，我还没写完。”
- “线歪，胆子正。”
- “我尾巴还能擦一下。”

### 代表抽象

- 生成
- 开始
- 草稿
- 试错
- 外化
- 第一版
- 先行动再理解

## 角色二：破橡皮

### 核心人格

慢、疑、冷静、爱挑刺。

破橡皮相信“看起来顺的东西，往往要擦一遍才露出真问题”。它不是消极，而是负责不让假清楚混过去。它越靠谱，身体越被消耗。

### 外形

- 不是完整橡皮块，而是残损橡皮碎块。
- 身体矮、钝、塌、厚，像一个小墩子。
- 粉色侧和灰白侧都必须残损。
- 粉色侧也不能画成完整圆角块。
- 粉色侧应有缺口、崩边、参差断面、灰痕。
- 灰白侧有铅笔灰、旧字印、擦不掉的黑痕。
- 两色之间有明显裂缝或不规则接缝。
- 日常可以有裂纹；高压时可以完全裂开。
- 两条短腿从底下勉强伸出来。
- 半睁眼，怀疑、疲惫、冷淡。
- 不要笑脸，不要大眼睛，不要干净完整。

### 状态机制

- 缺角状态：日常使用，表示旧、耗损。
- 裂缝状态：修改压力变大。
- 裂开状态：大修、推翻、认知崩塌。
- 胶带粘回状态：带着修补继续工作。

“裂开了”是高压状态标签，比“快散了”更准确。

### 动作

- 擦掉漂亮废话：去噪、删繁。
- 擦不干净，留下灰痕：修正不彻底、经验残留。
- 把自己擦出圆角：反复修改、自我消耗。
- 裂开：修改压力过载。
- 两半分别擦两边：分工处理、并行修正。
- 用断面堵漏洞：临时补救。
- 躺在错字上压住它：暂停错误扩散。
- 擦出一个窗口：让问题显形。
- 把乱稿擦成干净区域：重启、重新开始。

### 口头禅

- “这句有点虚。”
- “先别信。”
- “擦掉才看得见。”
- “你这是顺，不是对。”
- “别心疼废话。”
- “留点灰，记得疼。”
- “我裂开了，但我没同意。”
- “删掉以后，意思才出来。”
- “那叫蹭，不叫改。”

### 代表抽象

- 修正
- 怀疑
- 删除
- 复盘
- 证据
- 校准
- 去幻觉
- 停止规则

## 搭档关系

它们不是好朋友文具，而是互相嫌弃但离不开的创作搭档。

烂笔头看破橡皮：

- 太慢。
- 太扫兴。
- 老泼冷水。
- 动不动就擦。

破橡皮看烂笔头：

- 太莽。
- 太虚。
- 老制造垃圾。
- 把“写出来”误以为“想清楚”。

但它们共同成立：

- 没有烂笔头，永远没有第一版。
- 没有破橡皮，第一版永远在骗自己。
- 烂笔头负责把混沌变成痕迹。
- 破橡皮负责把痕迹擦到接近真实。

## 组合动作

- 烂笔头写一条歪路，破橡皮擦掉死路：试错路径。
- 烂笔头画框，破橡皮擦掉多余边框：建结构。
- 烂笔头写满一页，破橡皮只留三句话：提炼。
- 烂笔头断芯，破橡皮裂开：卡住但没放弃。
- 烂笔头把想法推出来，破橡皮把幻觉推回去：生成与校正。
- 破橡皮擦出一块空地，烂笔头重新下笔：重启。

## 适合表达的认知动作

先选认知动作，不画概念名词。

适合画：

- 第一版被推出去。
- 证据被拖回来。
- 假清楚被擦掉。
- 失败被分类。
- 死循环被闸门拦住。
- 改动被放进不同槽位。
- 好句子从废话里被擦出来。
- 空白被重新开出来。

不适合直接画：

- AI Agent
- Loop Engineering
- Prompt Engineering
- 反馈机制
- 评估系统
- 知识管理

这些概念需要先抽取为可视动作，再画成正文插图。

## 绘制原则

- 画正文插图，不画单页教程。
- 每张图只表达一个认知动作。
- 保持低微、笨拙、残损、认真。
- 不要儿童文具拟人。
- 不要商业吉祥物。
- 不要过度可爱。
- 不要完整干净的新文具。
- 不要让角色只是站在旁边。
- 不要把概念讲成 PPT 信息图。
